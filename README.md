# PyMojicks

The Official Emojicks Interpreter for Python.

## Examples

```js
>>> ✏ in 👉 📥("> ") 🔁 🔢⚓ 📤(in)⚓ 👵 This is a comment 👴
> 5
True
```

```js
>>> ✏️ my_func (✒️ str) ▶️
...    📤(str)⚓
... ◀️
>>> my_func("hello")⚓
hello
```

```js
✏️ my_func (✒️ prompt) ▶️
    ✏ str 👉 📥(prompt) 🔁 🔢⚓
    📤("You Wrote:")⚓
    📤(str)⚓
◀️
my_func("Enter Integer: ")⚓
```
